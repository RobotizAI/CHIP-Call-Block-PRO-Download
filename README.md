## 📱 O que é o CHIP Call Block PRO?

O CHIP Call Block PRO é um aplicativo Android criado para bloquear chamadas indesejadas usando os recursos oficiais do sistema Android.

Ele foi pensado especialmente para quem usa celular com 1 ou 2 chips e quer controlar melhor quais chamadas devem ser bloqueadas, pausadas ou permitidas.

## 🧠 Como o app funciona?

O app usa os recursos oficiais do Android para atuar como identificador e bloqueador de chamadas.

Para que o bloqueio funcione corretamente, o Android precisa permitir que o CHIP Call Block PRO seja definido como app padrão de identificação de chamadas e spam.

Sem essa permissão do sistema, o app pode abrir normalmente, mas não terá autoridade para bloquear chamadas em tempo real.

---

## ⚙️ Modos de proteção

O app permite configurar o comportamento de proteção para cada chip.

Por padrão inicial:

- CHIP 1: Desligado
- CHIP 2: Desligado
- Tema inicial: Verde Premium

Você pode alterar isso dentro do app, de forma simples e rápida.

## 📥 Como baixar o APK pelo GitHub

Após o projeto ser enviado para um repositório GitHub, o APK pode ser gerado automaticamente pelo GitHub Actions.

### Opção recomendada: baixar pelo GitHub Actions

1. Acesse o repositório do projeto no GitHub.
2. Clique na aba Actions.
3. Clique no workflow Build APK.
4. Abra a execução mais recente com o símbolo verde de sucesso.
5. Desça até a seção Artifacts.
6. Baixe o arquivo ZIP gerado.
7. Extraia o ZIP no seu computador ou celular.
8. Dentro dele estará o arquivo .apk.
9. Envie o APK para seu celular Android e instale.

## 📲 Como instalar no Android

1. Baixe o arquivo .apk.
2. Abra o arquivo no celular Android.
3. Se aparecer o aviso de segurança, toque em Configurações.
4. Ative a opção Permitir instalação de apps desta fonte.
5. Volte e toque em Instalar.
6. Abra o CHIP Call Block PRO.
7. Toque em Gerenciar proteção.
8. Defina o app como identificador de chamadas e spam, se o Android solicitar.

---

## 🛡️ Como ativar a proteção

Depois de instalar:

1. Abra o app.
2. Toque em Gerenciar proteção.
3. Defina o CHIP Call Block PRO como app padrão de identificação de chamadas e spam.
4. Volte para o app.
5. Escolha as configurações desejadas para cada chip.
6. Ative ou pause a proteção conforme sua necessidade.

## ⚠️ Limites importantes do Android

Algumas funções dependem das permissões e limitações do próprio Android.

Por isso:

- O app precisa ser definido como app padrão de identificação de chamadas e spam.
- Alguns fabricantes podem limitar recursos de bloqueio em segundo plano.
- A identificação exata do chip usado em chamadas pode variar conforme aparelho, operadora e versão do Android.
- O Android pode mudar telas, nomes e permissões dependendo da marca do celular.
- O app não promete acesso a recursos que o Android não disponibiliza oficialmente.

O CHIP Call Block PRO usa a melhor alternativa técnica possível dentro das regras do sistema Android.
